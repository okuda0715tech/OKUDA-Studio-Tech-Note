# 広告用語

## 広告主 ( Advertiser )

広告を作成して、どこかの広告枠にそれを表示する人や企業を示します。


## パブリッシャー

広告主のメッセージを表示する広告枠を供給します。


## ASP ( Affiliate Service Provider )

広告主とパブリッシャーを仲介する広告代理店を示します。
