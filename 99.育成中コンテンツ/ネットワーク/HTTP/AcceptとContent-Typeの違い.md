# Accept と Content-Type の違い

`Accept` ヘッダは、クライアントがサーバからどのようなデータタイプのデータを受け取れるかを示すものです。

`Content-Type` ヘッダは、クライアントとサーバ間でデータをやりとりする際に、  
データの送り手が受け取り手に、実際にどのようなデータタイプのデータを送ったかを知らせるものです。
