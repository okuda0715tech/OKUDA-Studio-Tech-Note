# 便利な初期設定

## ウィンドウの右端でテキストを折り返すか否か

ファイル -> ユーザー設定 -> 設定 -> 「editor word wrap」で検索

`Editor: Word Wrap` のプルダウンリストを `on` に変更すると、ウィンドウの右端でテキストを折り返す設定になる。
