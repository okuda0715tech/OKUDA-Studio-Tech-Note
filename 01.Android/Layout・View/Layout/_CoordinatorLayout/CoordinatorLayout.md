# CoordinatorLayout

## 概要

`CoordinatorLayout` は、 `FrameLayout` を高機能化したレイアウトです。ただし、 FrameLayout を継承しているわけではありません。


## gradle

```
dependencies {
    implementation "androidx.coordinatorlayout:coordinatorlayout:1.1.0"
}
```
