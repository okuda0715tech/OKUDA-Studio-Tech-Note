# 画像リソース編

# 画像の一覧をサムネイル（プレビュー）表示して、「どのファイルがどんな画像なのか」を瞬時に見分ける方法

クラスファイルなどの一覧を表示するタブの下あたりに `Resource Manager` というタブがある。  
これを開くと、 `Drawable` や `Color`  や `Layout` リソースをプレビュー表示してくれる。
