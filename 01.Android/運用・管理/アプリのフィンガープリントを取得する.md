# アプリのフィンガープリントを取得する

`Android Studio` の `Terminal` を開き、以下のコマンドを入力し、 `Ctrl + Enter` もしくは、  
`Ctrl + Shift + Enter` で実行する。

```
gradlew signingReport
```
