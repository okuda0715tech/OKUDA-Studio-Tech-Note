# N 進数

## 表記方法

| N 進数 | 例          | 表記ルール                     |
|--------|-------------|--------------------------------|
| 2 進数  | 0b111111010 | 先頭に 0b または 0B を付与する |
| 8 進数  | 0772        | 先頭に 0 を付与する            |
| 10 進数 | 123         | 特になし                       |
| 16 進数 | 0x1fa       | 先頭に 0x または 0X を付与する |


## 型

何進数であろうと、整数なら int 型、 long 型に代入することができます。
