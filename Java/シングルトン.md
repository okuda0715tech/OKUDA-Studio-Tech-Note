# シングルトン

```java
public final class MySingleton {

    private static final MySingleton sInstance = new MySingleton();

    public static MySingleton getInstance(){
        // ここでnullチェックしてしまうとスレッドセーフでなくなる。
        return sInstance;
    }

    // privateなコンストラクタ
    private MySingleton(){

    }
}
```
