# 無名クラスが直接複数のクラス_インターフェースを継承できる

```Kotlin
open class A(x: Int) {
  public open val y: Int = x
}

interface B {...}

// Kotlin では、無名クラスが複数のクラスとインターフェースを直接継承できる。
// この例では、クラス A とインターフェース B を直接継承している。
val ab: A = object : A(1), B {
  override val y = 15
}
```
