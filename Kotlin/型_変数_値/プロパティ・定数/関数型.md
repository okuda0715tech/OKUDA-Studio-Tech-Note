# 関数型

## 概要

関数の引数の数と型、戻り値の型を表す型を関数型と呼びます。


## サンプル

例えば、引数に Int, String をとり、戻り値に Boolean をとる関数型は以下のようになります。

```kotlin
(Int, String) -> Boolean
```


## 関数型に代入できるものとできないもの

**代入できるもの**

- ラムダ式
- 無名関数


**代入できないもの**

- 名前付き関数
