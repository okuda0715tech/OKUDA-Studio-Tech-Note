# 日付と時刻のサンプル

## 現在の日付と時刻を取得する

```sql
// 2022-04-26 04:52:22 ( UTC : 世界標準時 )
SELECT CURRENT_TIMESTAMP;
```


## 現在の日付を取得する

```sql
// 2022-04-26 ( UTC : 世界標準時 )
SELECT CURRENT_DATE;
```


## 現在の時刻を取得する

```sql
// 04:53:41 ( UTC : 世界標準時 )
SELECT CURRENT_TIME;
```
