# 表示サイズFAQ

## マージンも拡縮されるのか？

表示サイズを変更したとき、マージンが `dp` or `sp` で指定されていれば、
マージンも拡大 / 縮小されます。


## 左右に制約があり、マージン指定はなしの場合、 View の中心から拡大縮小されるのか？

はい。左右に制約があり、左右のマージン指定がない場合は、  
View の左右中心から、左右に向かってその View が拡大・縮小されます。
