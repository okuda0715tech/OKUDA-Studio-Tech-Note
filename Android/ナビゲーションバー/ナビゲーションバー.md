# ナビゲーションバー

## バックキーの制御

バックキーのタップイベントを受け取るには、Activityで`onBackPressed`コールバックをオーバーライドします。

```Java
public class MyActivity extends Activity {
  @Override
  public void onBackPressed() {
    if (...) {
      // カスタマイズした何らかの処理
    } else {
      // 通常のバックキー処理
      super.onBackPressed();
    }
  }
}
```
