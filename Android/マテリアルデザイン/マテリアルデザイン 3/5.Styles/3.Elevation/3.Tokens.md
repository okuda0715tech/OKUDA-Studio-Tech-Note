# トークン

## コンポーネント エレベーション

各マテリアル コンポーネントのデフォルトの高度を確認するには、 [高度一覧](https://m3.material.io/styles/elevation/tokens#df721a00-888e-4c5e-bfe1-5d905f167aaa) を確認してください。
