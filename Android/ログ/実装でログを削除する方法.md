# 実装でログを削除する方法

## 概要

R8 を使用せず、実装のみでログ出力処理を削除する方法を記載します。


## 実装方法

```java
private final static boolean LOG_OUTPUT = false;

If (LOG_OUTPUT) {
    Log.v("FOO_TAG", "Extra debug logging.");
}
```

static フィールドで、ログの出力条件を設定するだけです。

設定次第で、 if 文の判定が常に false になるため、  
その場合は、コンパイラの最適化で、ログ出力処理自体が  
プログラムから削除されます。
