# タッチイベントを独自領域で受け取る方法

## サンプル

```java
Path p = new Path();
Region r = new Region();

@Override
public void onDraw(Canvas canvas) {

    // Path の形を指定する
    p.moveTo(50, 50);
    p.lineTo(100, 50);
    p.lineTo(100, 100);
    p.lineTo(80, 100);
    p.close();      

    // 長方形を表すオブジェクトを生成する。
    RectF rectF = new RectF();
    // 長方形の４つの辺が、 Path の一番外側にピッタリくっつくように、４辺の位置を算出する。
    p.computeBounds(rectF, true);

    // Path の形を正確に Region r にコピーする。
    r.setPath(p, new Region((int) rectF.left, (int) rectF.top, (int) rectF.right, (int) rectF.bottom));
}   

public boolean onTouch(View view, MotionEvent event) {

    // Region の contains(x,y) メソッドで、 点(x,y)が Region に含まれているかを判定する。
    if(r.contains((int)event.getX(),(int) event.getY()))
        Log.d(TAG, "Touch IN");
    else
        Log.d(TAG, "Touch OUT");

    return true;
}
```
