# 画面回転時の状態保存

## IDを割り当てましょう

OSはViewの状態を自動的に保存してくれます。  
ただし、ViewにIDを割り当てていない場合は保存してくれないため、注意が必要です。


## BaseSavedStateを継承したSavedStateをネストされたstaticなクラスで作成します。

`Parcelable`はAmazonが段ボールに商品を詰めるように、データをパッケージ化するものです。
`Parcel`データの入ったパッケージそのものを指します。
