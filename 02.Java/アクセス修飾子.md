# アクセス修飾子

| アクセス修飾子              | アクセス可能範囲                    |
| -------------------- | --------------------------- |
| public               | 全ての場所からアクセス可能               |
| protected            | 同一クラス、同一パッケージ、サブクラスからアクセス可能 |
| 無指定(package private) | 同一クラス、同一パッケージからアクセス可能       |
| private              | 同一クラスからアクセス可能               |
