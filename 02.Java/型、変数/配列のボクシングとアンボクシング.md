# 配列のボクシングとアンボクシング

## ユーティリティ

```Java
public class BoxingAndUnboxing {

    public static int[] fromIntegerArray(Integer[] integerArray) {

        int[] intArray = new int[integerArray.length];

        for (int i = 0; i < integerArray.length; i++) {
            intArray[i] = integerArray[i];
        }

        return intArray;
    }

}
```
