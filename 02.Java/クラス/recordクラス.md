# recordクラス

## 概要

`record` クラスには以下の機能があります。

- フィールドが private final で強制されオブジェクトがイミュータブルになる
- コンストラクタ、 toString、equqls、hashCode 各メソッドが自動実装されコードがシンプルになるため、本当に必要な業務ロジックに注目できる


## Java 16 で導入

`record` クラスの機能は、 Java 16 で導入されました。
